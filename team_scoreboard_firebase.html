Firebase에 연결 중…

잠시만 기다려 주세요.

⚙️

Firebase 설정이 필요합니다

이 파일을 텍스트 편집기(메모장, VS Code 등)로 열어 아래 순서대로 설정하면 6대 기기가 실시간으로 연동됩니다.

1

Firebase 콘솔 접속

console.firebase.google.com → Google 계정으로 로그인 → 프로젝트 만들기

2

Realtime Database 활성화

왼쪽 메뉴 → 빌드 → Realtime Database → 데이터베이스 만들기 → 위치: 아시아(싱가포르) → 테스트 모드로 시작

3

웹 앱 등록 & 설정값 복사

프로젝트 개요 → 톱니바퀴 → 프로젝트 설정 → 하단 내 앱 → 웹(</>) → 앱 닉네임 입력 → 앱 등록
그러면 아래와 같은 코드 블록이 나타납니다. 값들을 복사하세요.

const firebaseConfig = {
  apiKey: "여기에 복사",
  authDomain: "여기에 복사",
  databaseURL: "여기에 복사 ← 필수",
  projectId: "여기에 복사",
  storageBucket: "여기에 복사",
  messagingSenderId: "여기에 복사",
  appId: "여기에 복사"
};
4

이 HTML 파일에 붙여넣기

이 파일을 텍스트 편집기로 열어 스크립트 상단 firebaseConfig 부분에 복사한 값을 붙여넣고 저장하세요.

5

각 기기에서 URL로 접속

교사용: ?mode=teacher  │  공용 TV: ?mode=display
팀 패드: ?team=1   ?team=2   ?team=3   ?team=4

💡 파일을 학교 서버나 GitHub Pages에 올리면 모든 기기가 같은 URL로 접속할 수 있습니다.
